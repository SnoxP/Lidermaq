# Guia de Contribuição - Lidermaq

Este projeto é mantido pela equipe da Lidermaq. Agradecemos seu interesse em contribuir! Para garantir um fluxo de trabalho organizado, siga as diretrizes abaixo:

## Como Contribuir

1. **Fork:** Faça um fork do repositório para sua conta pessoal.
2. **Branch:** Crie uma nova branch para sua funcionalidade ou correção:
   `git checkout -b feature/nome-da-sua-funcionalidade` ou `fix/nome-do-bug`.
3. **Desenvolvimento:** Faça as alterações necessárias no código.
4. **Commit:** Faça o commit das suas alterações seguindo boas práticas:
   `git commit -m 'feat: adiciona nova funcionalidade X'` ou `git commit -m 'fix: corrige erro Y'`.
5. **Push:** Envie para a sua branch no seu fork:
   `git push origin feature/nome-da-sua-funcionalidade`.
6. **Pull Request:** Abra um Pull Request para o repositório principal descrevendo claramente as alterações feitas.

## Código de Conduta
Seja gentil e respeitoso com todos os contribuidores.

Obrigado por ajudar a tornar a Lidermaq ainda melhor!
